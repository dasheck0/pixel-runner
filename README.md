# dasheck0-phaser3-template

This is an ES6 ready Phaser 3 template including game server.   

## Setup
To finalize the setup for your project do the following:

* Create a .env file on the base directory and add the following parameters:
```bash
PORT=
```

* Run npm install both in base directory and client directory
* Add appropriate license
* Delete this readme and add the reald content
* Start coding :)
